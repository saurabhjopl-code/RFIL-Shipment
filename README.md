# rfil-shipment

Documentation + static HTML for the Shipment Packing Workflow for RAJNANDINI FASHION INDIA LTD.

This repository contains a single-page HTML documentation that describes the Creator/Packer workflow, packing logic, and export rules.

## Branches (suggested)
- `main` - primary branch containing the source files.
- `gh-pages` or `docs` - optional branch/folder used for GitHub Pages publishing.

## How to publish (GitHub Pages)
1. Push this repo to GitHub under the name `rfil-shipment`.
2. In repository settings -> Pages, choose branch `main` and folder `/docs` **OR** use `gh-pages` branch.
3. If using `docs/` folder, move `index.html` into `docs/index.html`.

## Files included
- `index.html` - the HTML documentation (root or docs/index.html)
- `README.md` - this file
- `BRANCH_TREE.txt` - simple branch tree (below)
- `LICENSE` - MIT License
- `.gitignore` - basic ignore entries
- `assets/logo.svg` - placeholder RAJNANDINI logo (SVG)
